---
title: 'Agentic AI API Key'
---
An agentic AI API key is required to interact programmatically with agentic tools from LandingAI, including VisionAgent and Agentic Document Extraction.

View and get your agentic AI API key on the API Key page.

## Get the Agentic AI API Key
To navigate to the API Key page:

1. Open VisionAgent. 
2. Click the profile icon at the bottom left corner of the page.
3. Click your name and email address.
4. Click the API Key in the left navigation bar. The API Key page opens and shows your API key.
5. Click the Copy icon to copy your API key.
